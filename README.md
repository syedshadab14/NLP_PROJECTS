# Fake-News-Detection-NLP
End-to-end NLP project to detect fake news using TF-IDF and Logistic Regression with Gradio UI deployment.

# 📰 Fake News Detection using NLP

## 🚀 Overview
This project is an end-to-end Natural Language Processing (NLP) application that detects whether a news article is **Fake or Real**.

The model is trained using TF-IDF vectorization and Logistic Regression, and deployed using Gradio to provide an interactive user interface.

---

## 🎯 Features
- Text Cleaning & Preprocessing
- TF-IDF Vectorization
- Classification using Logistic Regression
- High Accuracy (~98%)
- Confusion Matrix & Evaluation Metrics
- Interactive UI using Gradio
- Real-time Predictions

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLP (TF-IDF)
- Gradio

---

## 📊 Machine Learning Workflow
1. Data Collection (Fake & Real News Dataset)
2. Data Cleaning (text preprocessing)
3. Feature Extraction (TF-IDF)
4. Model Training (Logistic Regression)
5. Model Evaluation (Accuracy, Confusion Matrix)
6. Model Deployment (Gradio)

---

## 📂 Dataset
- Source: Kaggle Fake & Real News Dataset
- Files Used:
  - Fake.csv
  - True.csv
